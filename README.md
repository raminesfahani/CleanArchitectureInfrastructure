- Event Bus

- Database Repo

    - Redis
- Event Store Database
- Logging
    - Elastic Search
    - Kibana
    - Apm
    - Serilog
- Message Brokers
    - RabbitMq
    - Kafka
    - Dapr
- Outbox Pattern
- Service Discovery and Healtch checking
    - Consul
    - Apm
- Docker Compose
